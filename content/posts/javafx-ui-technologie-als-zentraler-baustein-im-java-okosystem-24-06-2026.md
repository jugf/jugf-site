Aufgrund der Kundennachfrage, des Interesses aus der Wissenschaft und des wachsenden Bedarfs an fortschrittlichen Visualisierungen in AI-gestützten Anwendungen und Analysen führt Oracle die kommerzielle Unterstützung für JavaFX wieder ein.

Zur Entwicklung von Java Desktop-Anwendungen sind Funktionen wie UI-Steuerelemente, Diagramme, interaktive Medien, Webinhalte, Animationen, CSS-Styling, 2D- und 3D-Rendering, Rich Text und Property Binding erforderlich – und das alles in einem benutzerfreundlichen, plattformübergreifenden Programmierparadigma.

JavaFX bietet ein umfangreiches grafisches UI-Toolkit für die Entwicklung unternehmenskritischer Anwendungen, das zudem nahtlos mit Swing zusammenarbeitet. Seit der JavaOne 2026 Entwicklerkonferenz wurde JavaFX als strategische Java UI Technologie eingestuft und unterliegt der Oracle Java SE Roadmap mit dem dazugehörenden Support.

Im Vortrag werden neue Funktionen erläutert, die in den letzten Jahren entwickelt wurden und in JavaFX 27 eingeflossen sind. Es folgt ein Update zur RichTextArea, Demos und Beispielcode sowie ein Ausblick auf die künftigen Neuerungen.

_Wolfgang Weigend arbeitet als Master Principal Solution Engineer bei der ORACLE Global Services Germany GmbH im weltweiten Java Team. Er beschäftigt sich seit vielen Jahren mit Java, GraalVM, JavaFX und modernen UI-Technologien im Java-Ökosystem._
